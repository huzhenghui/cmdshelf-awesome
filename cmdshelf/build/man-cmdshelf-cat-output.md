**NAME**\
       cmdshelf-cat - Concatenate and print command(s).\
\
**SYNOPSIS**\
       **cmdshelf** **cat** \[\[\<remoteName\>:\]\<command\>\]\
\
**DESCRIPTION**\
       The  cat  subcommand  reads commands script files sequentially, writing\
       them to the standard output.\
