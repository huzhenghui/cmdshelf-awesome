**NAME**\
       cmdshelf-list - Show all registered commands.\
\
**SYNOPSIS**\
       **cmdshelf** **list** \[--path\]\
\
**DESCRIPTION**\
       The list subcommand lists all commands from remotes.\
\
**OPTIONS**\
       --path\
           Print absolute paths for each commands.\
